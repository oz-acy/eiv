簡易式畫像ビューアー "EIV"
  Ver 0.39 (2021.3.23 built)

簡單安易な畫像ビューアーである。
BMP、PNG、JPEGを表示できる。
但し制限はある。
  * 16bit BMP には對應せず. (別に問題ないやらうという安易な考へである。)
  * PNGのアルファカラー、透明色は無視。
  * 表示は、原寸大か、窗の大きさに收まる大きさか、何れかを選擇。

おまけ的要素
  * ディレクトリ内巡覽機能
    マウスでクリックすると次の畫像ファイルを表示する。
    右クリックすると左クリックの逆順に進む。
  * 壁紙設定機能
    環境變數 TMP (TMP が設定されてゐない場合は TEMP) で指定されたディレクトリに
    一時ファイルが作成される。
  * 24bit色から256色への減色機能
  * グレイスケール化
  * ファイル保存可能
    PNG保存時、256色畫像なら左上角の色を透明色に出來る。

インストール方法
  實行ファイル eiv.exe を適當なディレクトリに置けばよい。

アンインストール方法
  實行ファイルを削除すればよい。



實行環境
  作者は64bit版Windows 10で動かしてゐる。
  32bit版のOSでは動かないのではないかと思ふが、確かめてゐないので分からない。

ライセンス
  BSD二條項ライセンスとする。COPYINGのファイル名で文書を添附する。
